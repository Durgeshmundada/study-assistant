# AI Study Assistant

This project is a personal AI-powered study assistant built with Streamlit and LangChain. It helps users learn and revise topics, especially for DSA (Data Structures and Algorithms), by providing:

- **Concept Explanations**
- **Practical Examples**
- **Quiz Generation**
- **Notes Maker**
- **Flashcard Generator**

## Features
- Uses LangChain for LLM-powered responses
- Example generator helped me practice and understand DSA concepts
- Quiz and flashcards for self-testing
- Notes and explanations for revision

## Why this project?
I built this project to:
- Practice and learn DSA concepts interactively
- Explore and learn how to use LangChain for building LLM-powered apps
- Create a useful tool for exam preparation and revision

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your API keys to a `.env` file (see `.env.example`)
4. Run: `streamlit run study_assistant.py`

## What I learned
- DSA concepts and problem-solving by generating and practicing examples
- How to use LangChain for prompt engineering and LLM chains
- Building interactive web apps with Streamlit
